 **Matias Store**

O **Matias Store** é um projeto CRUD desenvolvido em **PHP** com banco de dados **MySQL**, permitindo o gerenciamento completo de produtos: cadastro, visualização, edição e exclusão.  
O projeto possui uma landing page (`index.php`) que serve como porta de entrada do sistema.

---

//Funcionalidades

-  Adicionar produtos
-  Editar produtos
-  Listar produtos
-  Excluir produtos
-  Armazenamento no MySQL

---

//Tecnologias Utilizadas

// Tecnologia / Função |
|-----------/--------|
| PHP       / Lógica de negócio e integração com o banco |
| MySQL     / Banco de dados relacional |
| HTML/CSS  / Interface do usuário |
