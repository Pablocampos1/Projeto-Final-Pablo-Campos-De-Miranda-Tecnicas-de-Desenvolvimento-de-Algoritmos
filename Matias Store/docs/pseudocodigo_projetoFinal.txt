Algoritmo Cadastrar_Cliente

Início

  1. Exibir formulário de cadastro de cliente.
  
  2. Aguardar o usuário preencher os campos:
        - nome
        - telefone
        - email
        - endereço (se existir)
        - demais campos do sistema

  3. Quando o usuário clicar no botão "Salvar":
        3.1 Receber os dados no arquivo salvar-cliente.php.
        
        3.2 Validar os dados:
              Se algum campo obrigatório estiver vazio:
                    Exibir mensagem de erro.
                    Encerrar processo.
              FimSe

  4. Conectar ao banco de dados.

  5. Executar comando INSERT para inserir os dados do cliente na tabela cliente.

  6. Se a inserção for bem-sucedida:
          Exibir mensagem: “Cliente cadastrado com sucesso”.
          Redirecionar para listar-cliente.php.
     Senão:
          Exibir mensagem: “Erro ao cadastrar cliente”.
     FimSe

Fim
